TrainTravel

Ce projet, TrainTravel à été réalisé dans le cadre de mon BUT informatique en première année (mai 2022). Ce site web permet la recherche et la réservation de billets de train. Le design n'était pas exigé, j'ai donc designé mon futur site sur Figma. J'ai ensuite développé le site internet.

Nous avions accès à une API gérée par le professeur pour nous permettre d'aller récupérer différentes informations (trains, horaires, gares, utilisateurs,...).
